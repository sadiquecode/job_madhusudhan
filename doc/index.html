TeachMe Project Installation Guide

Install Project and Setup Database

  1. Unzip the teachme.zip file in the public_html folder.
  2. Set up the database by creating a new database and user with appropriate permissions.
  3. Import the database from db/teachme_latest.sql.
  4. Update the database details in the .env file:
    5. DB_DATABASE=demo
DB_USERNAME=demo
DB_PASSWORD=demo
  5. Open the terminal and run composer update.

Setup Email

  1. Go to https://resend.com/overview and configure your domain.
  2. Add configuration details in the .env file:
    3. RESEND_API_KEY=re_ZvtBzSYs_CSn2aq67sKCD27ALzeDvdhm6

MAIL_MAILER=resend
MAIL_HOST=smtp.resend.com
MAIL_PORT=465
MAIL_USERNAME=resend
MAIL_PASSWORD=re_ZvtBzSYs_CSn2aq67sKCD27ALzeDvdhm6
MAIL_ENCRYPTION=null
MAIL_FROM_ADDRESS=<noreply@teachme.com>
MAIL_FROM_NAME=Resend
admin_mail=admin@gmail.com

Setup Stripe

  1. Go to https://stripe.com to generate keys, webhook, and two catalogs.
  2. Paste keys and webhook details in the .env file:
    3. STRIPE_KEY=
STRIPE_SECRET=
STRIPE_CURRENCY=AED
STRIPE_WEBHOOK_SECRET=
  3. Create two catalogs in Stripe: Tutor Premium and Student Premium with their respective prices and time durations.
  4. Copy the price_id from the package details.
  6. Paste the price_id in the dashboard, {url}dashboard/package for both student and tutor packages.

Logging in as Admin

Open the website in the browser and login as admin:

Note: These credentials are changeable. You can update them anytime from the profile settings.

Congratulations!

You have successfully installed TeachMe.
